### ShortCut

`ctrl + n`	     : Add new word  
`ctrl + e` 		 : Edit current search word or selected word  
`shift + delete` :  Delete current search word or selected word  
`ctrl + d` 		: bookmark word  
`ctrl + delete` : delete bookmark selected word on bookmark
